Detailed Description

template<typename T = milliseconds>
class Sine::General::Timer< T >

Sine::Timer is an automatic, scope-bound Timer.

Its constructor starts an internal clock and whose destructor outputs the duration since construction. As such, it times the length of a scope.

Template Parameters
T Unit measured (Sine::seconds, Sine::milliseconds, Sine::microseconds, Sine::nanoseconds).
